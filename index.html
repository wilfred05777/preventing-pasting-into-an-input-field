Preventing paste into an input field

This example demonstrates how to stop a user from pasting content into a HTML input form field using JavaScript. Check the HTML and JavaScript panes for more details or scroll to the bottom to find a code example.

ðŸ“Œ Works in modern browsers. The paste event is part of the Clipboard API. Check out the caniuse stats.

Code Example

<input type="text" id="paste-no-event" />
<script>
  const pasteBox = document.getElementById("paste-no-event");
  pasteBox.onpaste = e => {
    e.preventDefault();
    return false;
  };
</script>
